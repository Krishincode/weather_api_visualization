# Weather API Visualization

This project fetches live weather data for multiple cities using the **OpenWeatherMap API** and visualizes the information using **Python** and **Matplotlib**.

## Features
- Fetches current weather data (Temperature, Humidity, Feels Like) for any number of cities
- Dynamic user input for cities
- Visualizes Temperature, Feels Like, and Humidity using bar charts
- Handles invalid city names gracefully
